# Robotic Dog 3D  Design

## Overview
This repository contains the 3D CAD design of a robotic dog developed as part of the Smart Methods Mechanical Engineering Training Program.

The project demonstrates mechanical design skills, 3D modeling workflow, and preparation of CAD models for digital fabrication and 3D printing using Tinkercad.

## Project Description
The robotic dog model was designed using Tinkercad with a focus on:

- Mechanical structure design.
- 3D modeling techniques.
- Preparing the model for digital fabrication.
- Exporting the final model as an STL file for 3D printing.

## Model File
The repository includes the final exported STL file of the robotic dog design.

File Format:
- STL (.stl) — Ready for 3D printing and digital fabrication.

## Design Process
The design workflow included:

1. Developing the robotic dog concept.
2. Creating the 3D model using Tinkercad.
3. Refining the model structure and geometry.
4. Exporting the final design into STL format.

## Tools & Technologies

- Tinkercad for 3D modeling and design.
- 3D design techniques.
- STL file export.
- 3D printing preparation.

## Training Context
This project was completed as part of the Smart Methods Training Program – Mechanical Engineering Track, focusing on practical applications of digital design and modern engineering methods.
